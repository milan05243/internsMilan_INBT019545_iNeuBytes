# Task 1: Computer Vision Using CNN Models

## Objective
Implement and compare Traditional CNN and Customized CNN models for image classification on the CIFAR-10 dataset.

## Dataset
- CIFAR-10
- 60,000 RGB images
- 10 classes

## Models
- Traditional CNN
- Customized CNN

## Results
| Model | Test Accuracy |
|--------|--------------:|
| Traditional CNN | 73.85% |
| Customized CNN | 76.28% |

## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn
